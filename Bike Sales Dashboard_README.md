# Bike Sales Dashboard

![bike sales dashboard](<BIKE SALES.png>)
## 📘 Overview
The **Bike Sales Dashboard** analyzes customer demographics and purchasing behavior to identify patterns influencing bike purchases.

## 🎯 Objectives

- Examine how income, age, and commute distance affect bike purchases

- Compare purchase trends across gender, marital status, and education levels

- Identify target customer segments for marketing

## 📊 Key Insights

- **Higher income buyers:** Males ($60,134 average income)

- **Most active age group:** Middle-aged customers (318 purchases)

- **Commute trend:** Short-distance commuters (0–1 miles) show highest purchase count

## ⚙️ Filters

- Marital Status, Region, Education, Gender, Children

## 🧭 Tools Used

- Excel Pivot Charts  

- Interactive slicers  

- Line and bar charts for demographic analysis
